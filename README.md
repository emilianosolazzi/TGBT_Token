TGBT (Temporal Gradient Beacon Token) is a fixed-supply ERC-20 token on Arbitrum with an immutable hard cap of 2,000,000,000 tokens. The token features a permissionless proof-of-work mining system where anyone can mine TGBT using a CPU, with emissions anchored to Bitcoin block data. All administrative authority can be permanently removed via an ossification mechanism, making the token as immutable as Bitcoin.
